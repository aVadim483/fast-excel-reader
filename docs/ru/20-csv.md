## Возможности

* **Определение разделителя**: автоматическое или ручное указание разделителя колонок.
* **Широкая поддержка кодировок**:
    - UTF-8, UTF-16LE, UTF-16BE, UTF-32LE, UTF-32BE
    - Windows-1251, KOI8-R, ISO-8859-5
    - CP932, Shift_JIS, EUC-JP
    - Любая кодировка, поддерживаемая вашим окружением PHP.
* **Соответствие RFC 4180**: поддержка полей с кавычками и без, а также экранированных кавычек (удвоенных).
* **Многострочные поля**: обработка переносов строк внутри полей в кавычках.
* **Гибкий разбор**:
    - **Строгий режим**: строго следует RFC 4180.
    - **Толерантный режим**: более снисходителен к нестандартным CSV-файлам (например, кавычки внутри полей без кавычек).
* **Экономия памяти**: способен читать очень большие файлы построчно с помощью генераторов.
* **Дополнительные утилиты**: пропуск пустых строк, пропуск строк-комментариев, обрезка полей, произвольные ограничители и экранирующие символы.
* **Обработка BOM**: автоматическая обработка файлов с меткой порядка байтов (Byte Order Mark).

## Базовое использование

Проще всего открыть CSV-файл через метод `Excel::openCsv()`.

```php
use avadim\FastExcelReader\Excel;

$file = 'data.csv';
$csv = Excel::openCsv($file);

foreach ($csv->nextRow() as $rowNum => $row) {
    // $row — это простой числовой массив
    print_r($row);
}
```

## Продвинутое чтение

### Чтение с заголовками

Если в CSV есть строка заголовков, можно использовать метод `withHeader()`, чтобы значения первой строки стали ключами для последующих строк.

```php
$csv = Excel::openCsv($file);
$rows = $csv->withHeader()->nextRow();

foreach ($rows as $rowNum => $row) {
    // $row = ['ID' => '1', 'Name' => 'John', 'City' => 'New York']
    echo $row['Name'];
}
```

Имена столбцов можно задать и самостоятельно. Первая строка при этом всё равно пропускается, но имена
берутся из вашего списка, а не из её значений. Имена применяются по порядку столбцов, поэтому буквы
столбцов не нужны:

```php
$rows = $csv->withHeader(['id', 'name', 'city'])->readRows();
// $row = ['id' => '1', 'name' => 'John', 'city' => 'New York']
```

Более короткий список переименует только те столбцы, которые он покрывает; остальные сохранят имя из
строки заголовков.

### Чтение через генератор (экономно по памяти)

Метод `nextRow()` возвращает `\Generator`, что идеально для обработки больших файлов без загрузки их целиком в память.

```php
$csv = Excel::openCsv($file);
$generator = $csv->nextRow();

foreach ($generator as $row) {
    // Обработать каждую строку
}
```

### Чтение в массив

Если файл небольшой и нужны все данные сразу:

```php
$csv = Excel::openCsv($file);
$allRows = $csv->readRows();
```

## Опции конфигурации

В `openCsv()` можно передать массив опций или объект `CsvOptions`.

```php
use avadim\FastExcelReader\Csv\CsvOptions;

$options = [
    'delimiter' => ';',
    'enclosure' => '"',
    'encoding' => 'UTF-8',
    'trim_fields' => true,
    'skip_empty_lines' => true,
];

$csv = Excel::openCsv($file, $options);

// Другие способы задать опции:
$options = new CsvOptions($options)
    ->setDelimiter(';')
    ->setEnclosure('"')
    ->setEncoding('UTF-8')
    ->setTrimFields(true)
    ->setSkipEmptyLines(true)
;

$csv = Excel::openCsv($file, $options);
```

Доступные опции:

| Опция              | Тип      | По умолч.  | Описание                                                                     |
|--------------------|----------|------------|------------------------------------------------------------------------------|
| `mode`             | `string` | `'strict'` | Режим разбора: `'strict'` или `'tolerant'`                                   |
| `delimiter`        | `string` | `null`     | Разделитель колонок (например, `,`, `;`, `\t`). `null` или `'auto'` — автоопределение. |
| `enclosure`        | `string` | `"`        | Символ ограничителя поля.                                                    |
| `encoding`         | `string` | `null`     | Кодировка входного файла. `null` — автоопределение.                          |
| `double_quotes`    | `bool`   | `true`     | Обрабатывать ли удвоенные кавычки как экранированные.                        |
| `escape`           | `string` | `''`       | Экранирующий символ (например, `\`).                                         |
| `trim_fields`      | `bool`   | `true`     | Обрезать ли начальные/конечные пробелы у полей без кавычек.                  |
| `skip_empty_lines` | `bool`   | `true`     | Пропускать ли пустые строки.                                                 |
| `comment_prefix`   | `string` | `null`     | Символ(ы), обозначающие строку-комментарий (например, `#`).                  |

### Автоопределение разделителя

Если `delimiter` равен `null` или `'auto'`, ридер попытается определить разделитель, анализируя первые несколько строк файла.

### Поддержка кодировок

Ридер автоматически определяет большинство распространённых кодировок, включая UTF-8, UTF-16 и различные региональные кодировки вроде Windows-1251 или Shift_JIS. Также можно вручную указать любую кодировку, поддерживаемую функцией PHP `mb_convert_encoding`.

### Режимы разбора (строгий и толерантный)

* **Строгий режим (`'strict'`)**: выбрасывает ошибки, если CSV не строго следует RFC 4180 (например, если после закрывающей кавычки есть текст или внутри поля есть неэкранированные кавычки).
* **Толерантный режим (`'tolerant'`)**: пытается восстановиться и прочитать как можно больше данных при встрече с некорректной структурой CSV.

## Обработка ошибок

Можно задать собственный обработчик ошибок для управления проблемами разбора, что особенно полезно в `tolerant`-режиме.

```php
$csv = Excel::openCsv($file);
$csv->onError(function($code, $error, $line, $lineNo, $colNo) {
    echo "Error on line $lineNo, col $colNo: $error\n";
    echo "Line content: $line\n";
});
```

## Примеры

### Чтение файла с табуляцией (TSV)

```php
$csv = Excel::openCsv('data.tsv', ['delimiter' => "\t"]);
foreach ($csv->nextRow() as $row) {
    // ...
}
```

### Обработка файлов в кодировке Windows-1251

```php
$csv = Excel::openCsv('russian_data.csv', ['encoding' => 'Windows-1251']);
foreach ($csv->nextRow() as $row) {
    // ...
}
```

### Пропуск комментариев

```php
$csv = Excel::openCsv('config.csv', ['comment_prefix' => '#']);
foreach ($csv->nextRow() as $row) {
    // Строки, начинающиеся с #, будут проигнорированы
}
```
