## Получение правил проверки данных
Каждый лист в вашем XLSX-файле может содержать набор правил проверки данных (data validation). Чтобы получить их, достаточно вызвать `getDataValidations` у листа

```php
$excel = Excel::open($file);

$sheet = $excel->sheet();

$validations = $sheet->getDataValidations();
/*
[
  [
    'type' => 'list',
    'sqref' => 'E2:E527',
    'formula1' => '"Berlin,Cape Town,Mexico City,Moscow,Sydney,Tokyo"',
    'formula2' => null, 
  ], [
    'type' => 'decimal',
    'sqref' => 'G2:G527',
    'formula1' => '0.0',
    'formula2' => '999999.0',
  ],
]
*/
```

## Ширины колонок
Получить ширину конкретной колонки на листе:

```php
$excel = Excel::open($file);
$sheet = $excel->selectSheet('SheetName');

// Получить ширину колонки 1 (колонка 'A')
$columnWidth = $sheet->getColumnWidth(1);

echo $columnWidth; // Например: 11.85
```

## Высоты строк
Получить высоту конкретной строки на листе:

```php
$excel = Excel::open($file);
$sheet = $excel->selectSheet('SheetName');

// Получить высоту строки 1
$rowHeight = $sheet->getRowHeight(1);

echo $rowHeight; // Например: 15
```

## Информация о закреплённых областях
Получить информацию о закреплённых областях (freeze pane) листа:

```php
$excel = Excel::open($file);
$sheet = $excel->selectSheet('SheetName');

// Получить конфигурацию закреплённых областей
$freezePaneConfig = $sheet->getFreezePaneInfo();

print_r($freezePaneConfig);
/*
Пример вывода:
Array
(
    [xSplit] => 0
    [ySplit] => 1
    [topLeftCell] => 'A2'
)
*/
```

## Информация о цвете вкладки
Получить информацию о цвете вкладки листа:

```php
$excel = Excel::open($file);
$sheet = $excel->selectSheet('SheetName');

// Получить конфигурацию цвета вкладки
$tabColorConfig = $sheet->getTabColorInfo();

print_r($tabColorConfig);
/*
Пример вывода:
Array
(
    [theme] => '2'
    [tint] => '-0.499984740745262'
)
*/
```

## Информация об объединённых ячейках

Можно использовать следующие методы:

* ```Sheet::getMergedCells()``` -- Возвращает все объединённые диапазоны
* ```Sheet::isMerged(string $cellAddress)``` -- Проверяет, объединена ли ячейка
* ```Sheet::mergedRange(string $cellAddress)``` -- Возвращает диапазон объединения указанной ячейки

Например
```php
if ($sheet->isMerged('B3')) {
    $range = $sheet->mergedRange('B3');
}
```

## Подсчёт строк и колонок

Каждый лист содержит свойство ```dimension``` с диапазоном области, в которой записаны данные.
Если на листе заполнена только одна ячейка, там должен быть адрес только этой ячейки вида "B2",
иначе это диапазон вида "B2:E10".

Есть несколько методов, получающих данные из этого свойства:
* ```dimension()``` -- Возвращает размерность рабочей области по умолчанию из свойств листа
* ```countRows()``` -- Количество строк из dimension
* ```countColumns()``` -- Количество колонок из dimension
* ```minRow()``` -- Минимальный номер строки из свойств листа
* ```maxRows()``` -- Максимальный номер строки из свойств листа
* ```minColumn()``` -- Буква минимальной колонки из свойств листа
* ```maxColumn()``` -- Буква максимальной колонки из свойств листа

Но иногда свойство ```dimension``` содержит некорректную информацию.
Например, там может быть адрес только первой ячейки диапазона данных или только последней.
В таких случаях можно использовать методы, которые сканируют весь лист и подсчитывают фактическое
количество строк и колонок с данными на листе.

ВАЖНО: эти методы работают медленнее методов, использующих свойство ```dimension```

* ```actualDimension()``` -- Возвращает размерность фактической рабочей области
* ```countActualRows()``` -- Подсчёт фактических строк на листе
* ```minActualRow()``` -- Минимальный фактический номер строки
* ```maxActualRow()``` -- Максимальный фактический номер строки
* ```countActualColumns()``` -- Подсчёт фактических колонок на листе
* ```minActualColumn()``` -- Буква минимальной фактической колонки
* ```maxActualColumn()``` -- Буква максимальной фактической колонки
